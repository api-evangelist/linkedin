# LinkedIn (linkedin)
LinkedIn is a professional networking platform providing APIs for consumer integrations (Sign In, Share, Verified on LinkedIn), marketing solutions (ad campaigns, audiences, conversions, analytics), talent solutions (job posting, recruiter system connect), learning solutions (activity reports, content), sales navigator (CRM sync, display, analytics), compliance (message archiving), and regulatory data portability.

## Features

| Name | Description |
|------|-------------|
| Sign In with LinkedIn | OpenID Connect-based authentication enabling users to sign in to third-party applications with their LinkedIn credentials. |
| Share on LinkedIn | Enable users to share content from your application directly to their LinkedIn feed. |
| Verified on LinkedIn | Display verified professional credentials and certifications from LinkedIn on third-party platforms. |
| Campaign Management | Programmatically create, manage, and optimize B2B advertising campaigns with targeting and budget controls. |
| Audience Insights | Access aggregated audience demographics and firmographic data for ad targeting and market research. |
| Conversions API | Send online and offline conversion events for attribution and campaign optimization. |
| Lead Gen Forms | Collect leads directly from LinkedIn ads with pre-filled forms using member profile data. |
| Job Posting | Programmatically publish and manage job listings on LinkedIn through the Talent Solutions API. |
| Recruiter System Connect | Integrate applicant tracking systems with LinkedIn Recruiter for seamless candidate management. |
| Learning Activity Reports | Retrieve learner engagement, completions, and activity data from LinkedIn Learning. |
| Compliance Archiving | Archive and monitor LinkedIn messages and activities for regulatory compliance in regulated industries. |
| Data Portability | Access member and organization data portability endpoints for Digital Markets Act compliance. |
